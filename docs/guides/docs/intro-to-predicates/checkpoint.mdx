## 构建 Predicate

要格式化你的合约，请执行以下命令：

<TestAction
id="format-predicate"
action={{
  name: 'runCommand',
  commandFolder: 'guides-testing/multisig-predicate/predicate'
}}
/>

```sh
forc fmt
```

要获取 predicate 根哈希，进入 predicate 文件夹并运行：

<TestAction
id="build-predicate"
action={{
  name: 'runCommand',
  commandFolder: 'guides-testing/multisig-predicate/predicate'
}}
/>

```sh
forc build
```

你的 predicate 根哈希应该是：

```sh
0x9cdce04cdb323e5982bbd0c07f667c6ea2b97781a8ce6f3e2d96c0e1b5acde73
```
就是这样！你已经创建了你的第一个**无状态**的去中心化应用，而且我们甚至不需要部署它！
