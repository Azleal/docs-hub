## 构建合约

以下是精简后的说明：

要格式化你的合约，请执行以下命令：

<TestAction
id="format-contract"

  name: 'runCommand',
  commandFolder: 'guides-testing/fuel-project/contract'
}}
/>

```sh
forc fmt
```

要编译你的合约，请转到合约文件夹并运行：

<TestAction
id="build-contract"
action={{
  name: 'runCommand',
  commandFolder: 'guides-testing/fuel-project/contract'
}}
/>

```sh
forc build
```

恭喜！你已成功用 Sway 编写了一个完整的合约！

编译后，系统将自动生成 `abi.json`、`storage_slots.json` 和 `contract.bin`。你可以在以下目录找到这些文件：

```sh
contract/out/debug/*
```

## 部署合约

有关部署此合约的详细步骤，请参阅官方 Fuel 开发人员计数器 DApp 指南：
[部署合约](/guides/counter-dapp/building-a-smart-contract/#deploy-the-contract)

要部署，请使用以下命令，如果你已经设置了 forc-wallet 并且在你的账户中有测试网资金。如果没有，请按照上面的说明进行操作。

```sh
forc deploy --testnet
```

部署后，你将能够在 `contract/out/deployments` 文件夹中找到你的合约 ID。你将需要这个 ID 进行前端集成。
