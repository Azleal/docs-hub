# 恭喜您完成了Sway入门指南！🎉

遇到问题了吗？一个有用的初始步骤是将您的代码与存储库对齐，并解决任何差异。查看项目的存储库[这里](https://github.com/FuelLabs/intro-to-sway/tree/main)。🔍

对您的成就感到兴奋吗？在Twitter上与我们分享[@fuel_network](https://twitter.com/fuel_network)。
通过这样做，您可以获得专属开发者社区的访问权限，收到我们即将举办的Fuel晚宴的邀请，甚至可以获得有关项目的内部信息。
留意惊喜！👀
