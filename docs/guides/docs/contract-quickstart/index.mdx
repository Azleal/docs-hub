## 生成一个计数器合约

运行以下命令来生成一个 Sway 计数器合约：

```sh
forc template --template-name counter counter-contract
```

合约会在 `src/main.sw` 文件中。

  file="../counter-dapp/building-a-smart-contract.mdx"

## 构建合约

要构建合约，进入 `counter-contract` 文件夹：

```sh
cd counter-contract
```

然后运行 `forc build` 命令：

```sh
forc build
```
