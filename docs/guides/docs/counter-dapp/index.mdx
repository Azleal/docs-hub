# 计数器 Dapp

本指南包括以下逐步说明：

- 用 Sway 编写计数器智能合约
- 用 Rust 编写测试
- 部署到 Fuel 的测试网
- 构建前端
- 集成钱包

在开始之前，了解文档中将使用的术语及其相互关系可能会有所帮助：

- **Fuel**：Fuel 区块链。
- **FuelVM**：为 Fuel 提供动力的虚拟机。
- **Sway**：为 FuelVM 设计的领域特定语言，受 Rust 启发。
- **Forc**：Sway 的构建系统和包管理器，类似于 Rust 的 Cargo。
